# Synthesis Extracts

This directory stores synthesized pattern groups generated by `dtp synthesize`.

Synthesis docs combine pattern candidates, lessons, and decisions into a reusable planning record. They are meant to help `dtp draft` say, "this scope can reuse a proven pattern," without pretending a new client should receive copied code blindly.

Every synthesis doc starts with `promoted: false` and `private_review_required: true`.
