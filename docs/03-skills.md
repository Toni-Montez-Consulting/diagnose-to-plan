# Skills

Draft-time Skills in `skills/`:

- `voice`
- `pricing`
- `sow`
- `build-patterns`

`voice`, `pricing`, and `sow` remain Toni-authored placeholders. `build-patterns` points drafts at reviewed synthesis docs.

Operator Skills in `practice-os/skills/` are reusable practice-process assets for Diagnose, roadmap planning, client context, proposal drafting, handoff runbooks, COI screening, and redaction. These are validated by `dtp practice doctor`, not loaded into every draft by default.

Use `practice-os/templates/activation-routing-map.md` when a prompt could route to `tm-skills`, a DTP operator Skill, a Practice OS template, a proof/COI gate, a roadmap story, or a parked automation path.
