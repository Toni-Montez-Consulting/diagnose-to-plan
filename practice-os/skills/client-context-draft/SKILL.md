# Purpose

Create the operating context an agent or future maintainer needs for one engagement.

# Data Allowed

Client-private information may appear only in the private engagement folder.

# Data Prohibited

Do not copy client-private details into `practice-os/`, public docs, or reusable Skills.

# Output Format

Sections: business, bottleneck, owner constraints, systems of record, approval rules, voice notes, data sensitivity, current phase, open decisions, do not do.

# Human Approval Trigger

Ask before adding customer names, financial specifics, or screenshots.
