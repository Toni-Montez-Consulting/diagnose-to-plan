# Redaction Policy

Redact before external sharing:

- client identity unless named permission exists
- customer identities and employee names
- vendor names unless public and approved
- financial specifics
- internal screenshots
- credentials and tokens
- proprietary process details
- Microsoft-sensitive context

Anonymize with permission:

- industry
- region
- company size
- workflow category
- metric category

Named external use requires explicit written approval.
