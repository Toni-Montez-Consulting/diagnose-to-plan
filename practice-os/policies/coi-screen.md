# COI Screen

Run before proposal or serious scoping.

Collect:

- prospect legal name and website
- industry and location
- business model
- voluntarily disclosed customer categories
- tools and platforms involved
- whether Microsoft products, purchasing, Azure, Copilot, M365, Purview, GitHub Enterprise, or vendor selection are central
- whether the work overlaps Toni's day-job area

Verdicts:

- `pass`: proceed
- `flag`: pause, narrow scope, or get written guidance
- `block`: refer out

Never store internal Microsoft customer lists, account context, partner context, roadmaps, policies, code, or deal information.
