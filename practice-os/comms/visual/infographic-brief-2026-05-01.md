# Infographic Brief: The Operator System

## Goal

Create a one-page visual that explains the practice at a glance without hype.

## Panel 1: The Operator

Visual: one person at the center with work orbiting them.

Copy: "The real unit is the operator: the owner, spouse, assistant, coach, or
admin carrying the business day to day."

## Panel 2: The Bottleneck

Visual: texts, calendar, spreadsheet, payment status, customer question, and
memory all feeding one person.

Copy: "The business is not missing one magic tool. Too much work lives in one
person's head."

## Panel 3: The System

Visual: intake, workflow, data, assistant, dashboard, runbook, review rhythm.

Copy: "We diagnose the pressure, build the first useful system, and make the
operating path visible."

## Panel 4: The Handoff

Visual: operator using the system without Toni in the middle.

Copy: "The proof is the handoff. The operator can run it, change safe parts,
and know what not to touch."

## Panel 5: The Compounding Practice

Visual: engagement -> lesson -> skill/template -> fixture/eval -> next
engagement.

Copy: "Every engagement improves the Business Brain after review. The system
gets sharper without becoming SaaS."

## Style Notes

- Plain, serious, human.
- Avoid futuristic robot imagery.
- Use real artifact shapes: runbook, checklist, dashboard, note, proof packet.
- Keep public proof placeholders abstract until permission gates pass.
