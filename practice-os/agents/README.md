# Practice OS Agent Roles

These are operating modes for agents working inside DTP. They are not autonomous
employees and they do not authorize external writes, sends, billing actions,
contracting, publishing, or production changes.

Each role writes repo artifacts, marks missing data plainly, and escalates when
human judgment is required.
