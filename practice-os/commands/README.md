# Practice OS Command Contracts

These are command contracts for Business Brain behavior. They are not shell
commands yet unless the CLI implements them. Each contract defines the inputs,
outputs, guards, acceptance criteria, and fixtures future agents must follow.

Agents should update these contracts before implementing automation. Do not
create a second command tree outside DTP.
