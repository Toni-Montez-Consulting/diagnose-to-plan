# Agentic Performance Gap Review

Use this review when an agent session, roadmap change, skill update, or automation idea exposes a possible gap in routing, memory, validation, research, safety, or learning.

This template checks whether the operating system performed well, not just whether the work got done. It is required by `dtp practice doctor` because agentic process gaps can become repeated execution failures if they stay in chat.
